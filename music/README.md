# Background Music

Place your Genshin Impact-style music files here (.mp3 or .ogg).

Recommended tracks:
- `main.mp3` — Main background music (plays on landing page and during quiz)
- `result.mp3` — Result reveal music (optional, plays on result page)

## Where to get Genshin Impact OST
- Official OST playlists on YouTube/Bilibili
- HoYoverse official releases

## Supported formats
- MP3 (.mp3)
- OGG (.ogg) — better compression
- WAV (.wav) — larger file size

The app will auto-detect any music files in this folder.
If no files are found, users can drag-and-drop their own audio file onto the music button.
